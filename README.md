# Markdown Inline Editor for VS Code

WYSIWYG inline markdown editor that hides syntax markers and shows formatted content as you type.

![Screenshot of Markdown Inline Editor in action](https://github.com/domdomegg/markdown-inline-editor-vscode/assets/4953590/112da46d-214b-4d2c-8f13-c269c8d040cd)

## Features

- **Hidden syntax markers** - See formatted text instead of `**bold**` or `~~strike~~`
- **Styled headings** - Headers are visually distinct and easy to scan
- **Clean editing** - Focus on content, not markup
- **Pure markdown** - File remains standard markdown for AI tools and scripts
